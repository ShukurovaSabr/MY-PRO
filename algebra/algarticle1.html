Линейное уравнение

Линейное уравнение — это уравнение первой степени с одной переменной. Оно записывается в общем виде как:

ax + b = 0,

где a и b — заданные числа, причём a ≠ 0, а x — переменная, значение которой нужно найти.

Решение линейного уравнения

Решение заключается в нахождении такого значения x, при котором уравнение становится верным. Чтобы решить уравнение, нужно выразить x через остальные известные числа.

Пошаговый метод решения:

  1. Переносим свободный член (число без x) на правую сторону уравнения, меняя знак. Если уравнение было ax + b = 0, оно станет ax = -b.
  2. Делим обе части уравнения на коэффициент при x, то есть на a, чтобы получить x = -b / a.

Таким образом, формула решения линейного уравнения — x = -b / a.

Пример

Рассмотрим уравнение 3x + 6 = 0.

Шаг 1: Переносим 6 направо с противоположным знаком:

3x = -6.

Шаг 2: Делим обе части на 3:

x = -6 / 3 = -2.

Ответ: x = -2.

Особенности и случаи

  • Если коэффициент a равен 0, то уравнение принимает вид b = 0. Если при этом b = 0, то уравнение верно при любом x — решений бесконечно много.
  • Если a = 0, но b ≠ 0, уравнение не имеет решений.

Практическое применение

Линейные уравнения широко применяются в математике, физике, экономике и других науках для моделирования простых зависимостей между величинами. Например, если скорость постоянна, путь и время связаны линейным уравнением.
