Логарифмы

Логарифм — это операция, обратная возведению в степень. Логарифм числа по основанию показывает, в какую степень нужно возвести основание, чтобы получить это число.

Обозначается как loga(b), что читается как логарифм числа b по основанию a.

Определение логарифма

Если aⁿ = b, где a > 0, a ≠ 1, b > 0, то loga(b) = n.

Свойства логарифмов

  • loga(xy) = loga(x) + loga(y)
  • loga(x/y) = loga(x) - loga(y)
  • loga(xⁿ) = n · loga(x)
  • loga(a) = 1
  • loga(1) = 0

Изменение основания логарифма

Если нужно вычислить логарифм по другому основанию, используется формула:

loga(b) = logc(b) / logc(a)

Примеры

Вычислим log2(8). Поскольку 2³ = 8, логарифм равен 3.

Вычислим log10(1000). Поскольку 10³ = 1000, логарифм равен 3.

Применение

Логарифмы используются в различных областях: в науке, технике, информатике для решения уравнений, связанных с ростом, затуханием, а также для вычислений с большими числами и обработки данных.
